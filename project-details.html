프로젝트 주제

  • 'UDADA'웹의 목적은 단순히 데이터를 입출력 하는 것이 아니라 받은 데이터를 가공, 분석하여 사용자에게 보다 유용하고 가치있는 데이터를 보여 주는 것입니다.
  • 사용자의 섭취 음식, 한 운동, 키와 체중 정보를 받아 여러 정보를 제공합니다.
  • 다이어트와 관련된 데이터(칼로리, BMI)를 제공하고 음식 데이터를 이용해 레시피를 작성 하거나 운동 데이터를 이용해 목표를 설정하고 도전 할 수도 있습니다.
  • 이러한 데이터와 컨텐츠를 바탕으로 사용자가 다이어트 할 때 성공적인 다이어트를 할 수 있도록 도와줍니다.

개발 개요

  • Lombok, HikariCP, 를 사용한 효율성 있는 개발, Spring Security 사용한 사용자 비밀번호 암호화
  • ERDCloud를 사용한 직관성 있는 테이블 시나리오 및 관계 틀 형성
  • 사용자에게 받은 데이터를 바탕으로 여러가지 유용한 데이터 제공
  • 사설 호스팅이 아닌 ORACLE CLOUD를 사용한 CentOS 서버 구축 및 호스팅, SSL 인증

분담업무 페이지

  • 데이터 유/무에 따라 다른 메세지 출력
  • MyBatis 동적쿼리를 이용한 다양한 검색
  • Javascript 를 활용한 폼 입력 검증, 직관성 있는 검증 확인
  • AJAX를 사용한 비동기 DB조회 및 가져오기
  • 입력받은 데이터를 사용하여 Chart.js, FullCalendar 라이브러리로 사용자에게 유용한 데이터 제공
