프로젝트 주제

  • '올림픽나우' 프로젝트의 목적은 도쿄 올림픽 기간, 네이버 뉴스 기반으로 어떤 종목과 선수가 얼마나 언급됐는지 분석하고 차트로 보여주는 것입니다.
  • 도쿄 올림픽의 생각보다 높은 관심에 마케팅 활용성이 높아 졌고, 마케팅 활용을 위해 어떤 종목과 선수를 활용해야 하는지 보여줍니다.
  • 네이버 뉴스 API 를 사용해 오픈소스 활용 능력을 높이고 Javascript, Ajax 사용 능력 향상, Python-django 에 대한 이해를 높혔습니다.

개발 개요

  • Python-django 를 활용한 간결하고 효율적인 개발
  • 다양한 Python 모듈, Javascirpt 라이브러리 사용
  • 네이버 뉴스 API를 사용해 가치있는 데이터 수집
  • 수집, 정제한 데이터 시각화

개발 진행

  • 네이버 API를 사용해 올림픽 관련 기사 제목 데이터 추출
  • 기사제목 데이터를 Konlpy과 Komoran 오픈소스를 사용해 형태소 분석
  • 수집한 종목명, 선수명을 사용자 사전에 등록하고 종목명, 선수명만 추출
  • collections 모듈로 종목명, 선수명 언급 횟수 분석, json 저장
  • cx_oracle 모듈을 사용해 Oracle Cloud DB에 저장
  • 데이터를 주고 받기위해 Ajax, DB 조회 쿼리 작성

구현 페이지

  • Ajax를 사용한 기간 검색, 기간별 분석 결과 확인
  • 종합 - 확인 결과로 워드클라우드 이미지 생성 가능
  • 종목, 선수 - 기간별 라인차트로 변화 추이 확인 가능
